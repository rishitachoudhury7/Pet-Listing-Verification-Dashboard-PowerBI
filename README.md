# Pet-Listing-Verification-Dashboard-PowerBI
Interactive Power BI dashboard analyzing pet service listings across cities. Includes KPIs like Total Listings, Verification Rate, Avg Verification Days, and Source performance. Uses DAX measures, calculated columns, and slicers to explore Category, City, Status, and monthly submission trends.
This project presents an interactive Power BI dashboard analyzing pet service listings across multiple cities.
The dashboard focuses on listing performance, verification workflow, and source effectiveness.
It uses DAX measures, calculated columns, and slicers to generate actionable insights.
Designed to demonstrate data modeling, KPI creation, and business-oriented visualization skills.
